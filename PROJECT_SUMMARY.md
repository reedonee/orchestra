## 🚀 Getting Started (3 Simple Steps)

### Step 1: Run Setup Script
```powershell
cd c:\Users\hp\Desktop\red\create\Orchestra
.\setup.bat
```
**Or manually:**
```powershell

npx react-native-windows-init --overwrite --language cpp --useWinUI3
```

### Step 2: Wait for Completion
The script will:
1. Install npm dependencies (~2-3 minutes)
2. Initialize React Native Windows with C++/WinUI3 (~1-2 minutes)
3. Verify Visual Studio and required tools

### Step 3: Run the App
```powershell
npm run windows
```

**Expected Result:**
- ✅ Metro bundler starts
- ✅ Windows app window opens
- ✅ React Native welcome screen displays

## ❌ Common Issues & Solutions

### "npm: command not found"
→ Install Node.js from [nodejs.org](https://nodejs.org/)

### "Cannot find Visual Studio"
→ Install VS 2022 with "Desktop development with C++" workload

### "Windows SDK not found"
→ Open Visual Studio Installer → Check Windows SDK in workloads

### "Metro bundler won't start"
→ Run: `npm run start -- --reset-cache`

→ Or kill process on port 8081: `netstat -ano | findstr :8081`

### Build fails with C++ errors
→ Delete `windows/build` folder and rebuild

→ Update Visual Studio to latest version
