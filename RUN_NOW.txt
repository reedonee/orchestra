## Copy This If You Want Windows

Save this as a file and run as Administrator in PowerShell:

```powershell
# Enable Developer Mode
reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\AppModelUnlock" /v AllowDevelopmentWithoutDevLicense /t REG_DWORD /d 1 /f

# Enable Long Paths
reg add "HKLM\SYSTEM\CurrentControlSet\Control\FileSystem" /v LongPathsEnabled /t REG_DWORD /d 1 /f

# Install .NET SDK 8.0
winget install Microsoft.DotNet.SDK.8 -e -s winget

# Install Visual Studio 2022 Community
winget install --id Microsoft.VisualStudio.2022.Community -e

# Configure Visual Studio for React Native Windows
PowerShell -ExecutionPolicy Bypass -File "$PSScriptRoot\node_modules\react-native-windows\scripts\rnw-dependencies.ps1" -Install

Write-Host "Done! Run: npm run windows" -ForegroundColor Green
```

**Time:** 60 minutes (mostly waiting for downloads/installs)
